# Advent of Code - 2018

Solutions to [Advent of Code - 2018](https://adventofcode.com/2018/) written
in Rust (1.30).

## Rust

Install [rustup](https://rustup.rs/).

## Run

Run the unit tests:

```
cargo test
```
